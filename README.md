3. desktop - 1120px и выше;

1. mobile - 280px;

### SASS

Для стилизации своих компонентов создавайте файл в src/sass/components.
Всякое вспомогательное (миксины, функции и т.д.) создавать в папке helpers.
После импортируя это в main.scss;
